# 🤖 Agente Anti-Cancelación

## Descripción

Agente inteligente basado en IA generativa diseñado para reducir la cancelación de servicios mediante el análisis de intención del cliente, sentimiento y generación de estrategias personalizadas de retención.

El agente analiza una conversación con un cliente que desea cancelar su servicio, identifica las causas del problema y propone alternativas para mejorar la experiencia.

---

# 🎯 Objetivo del proyecto

Construir un agente conversacional capaz de:

- Detectar intención de cancelación.
- Analizar sentimiento del cliente.
- Evaluar riesgo de pérdida del cliente.
- Generar una estrategia personalizada de retención.
- Mantener una conversación empática.

---
